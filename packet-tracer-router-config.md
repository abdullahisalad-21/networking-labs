# Packet Tracer Router Configuration

## Overview
These are the foundational Cisco IOS commands used to configure routers in Packet Tracer.

---

## 1. Entering Configuration Mode
enable configure terminal

---

## 2. Setting Hostname
hostname Router1

---

## 3. Configuring Interfaces
interface g0/0 ip address 192.168.1.1 255.255.255.0 no shutdown exit

---

## 4. Configuring Default Gateway (on PCs)
ipconfig /defaultgateway 192.168.1.1

---

## 5. Saving Configuration
write memory
