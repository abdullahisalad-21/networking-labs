# OSI Model and Networking Protocols

## Overview
The OSI model explains how data moves across a network. Understanding it helps diagnose issues at each layer.

---

## 1. Layer 1 — Physical
- Cables  
- Connectors  
- Wi-Fi signals  

Tools:
cable check, NIC lights

---

## 2. Layer 2 — Data Link
- MAC addresses  
- Switches  
- ARP  

Commands:
arp -a

---

## 3. Layer 3 — Network
- IP addresses  
- Routing  
- Subnets  

Commands:
ipconfig /all tracert

---

## 4. Layer 4 — Transport
- TCP  
- UDP  

---

## 5. Layer 7 — Application
- DNS  
- HTTP  
- DHCP  

Commands:
nslookup ping domain.com

---

- How each OSI layer works  
- How tools like ping, tracert, and nslookup map to OSI layers
