# Networking Labs for IT Support

This repository contains hands-on networking practice, troubleshooting workflows, and Packet Tracer router configuration. It covers everything from basic commands to OSI model analysis.

## Skills Demonstrated
- IP addressing and subnet understanding
- DNS troubleshooting (nslookup, ipconfig /all)
- Connectivity testing (ping, tracert)
- Cisco Packet Tracer router configuration
- OSI model and protocol analysis
- Physical networking (racks, patch panels, wiring closets)
